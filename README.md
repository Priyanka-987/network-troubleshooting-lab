# Network Troubleshooting Lab

This repository contains practical network and infrastructure troubleshooting scenarios based on common operational issues encountered in cloud-native and telecommunications environments.

## Scenarios

| Scenario                   | Status |
| -------------------------- | ------ |
| BGP Neighbor Down          | ✅      |
| Gateway Unavailable        | ✅      |
| ECFE Config Stale          | ✅      |
| High Root Disk Utilization | ✅      |
| OAM Killer Investigation   | ✅      |
| Connectivity Testing       | ✅      |

## Objective

The goal of this repository is to document structured troubleshooting methodologies, investigation procedures, root cause analysis, and resolution steps for common infrastructure and networking issues.

## Skills Demonstrated

* Network Troubleshooting
* Root Cause Analysis
* BGP Investigation
* Linux Administration
* Kubernetes Troubleshooting
* Infrastructure Operations
* Incident Investigation
* Production Support Methodology

## Disclaimer

All examples are based on lab environments and do not contain any customer, production, or proprietary information.
